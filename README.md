## Description

[Nest](https://github.com/nestjs/nest) Template with automated dependency updates.

## Installation

```bash
$ pnpm install
```

## Environment Variables
Create a `.env` file and fill in all the credentials/variables mentioned in `.env.example`


## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```
